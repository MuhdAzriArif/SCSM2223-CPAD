Glossary

HTML
The standard language used to create the structure and content of a webpage.
CSS
The language used to design and style the visual layout, colors, and fonts of a website.
DOM
A programming interface for web documents. It represents the page so that programs (like your JavaScript) can change the document structure, style, and content dynamically.
Full-Stack Development
Development that involves both the "Frontend" (the user interface you see) and the "Backend" (the server, database, and logic that powers the site).
